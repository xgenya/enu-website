# 欢迎来到 ENU Website

**ENU Website** 是一个使用 [Next.js](https://nextjs.org) 和 [Nextra](https://nextra.site) 构建的文档与搜索平台。

## 特性

- 📚 **文档系统** — 基于 MDX 的结构化文档，支持 Markdown 和 React 组件
- 🔍 **全文搜索** — 内置 Pagefind 驱动的快速搜索功能
- 🌙 **深色模式** — 支持亮色/深色主题切换
- 📱 **响应式设计** — 完美适配桌面端和移动端
