# 服务器简介

## 服务器信息

ENU 是一个以原版生存为核心的 Minecraft 生电服务器，不添加任何影响游戏机制的插件或模组。

| 项目 | 内容 |
|------|------|
| 服务器版本 | Java 版 1.21.1 |
| 服务器 IP | `eunoia.ink` |
| 语音频道 | [Kook 频道](https://kook.vip/SbBtk5) |

**服务器特点：**
- 无传送、无死亡不掉落、无菜单
- 无其他非原版的娱乐向内容
- 纯原版生存体验

---

## 常用指令

| 指令 | 说明 |
|------|------|
| `/reg 密码 密码` | 首次进入服务器注册账号 |
| `/server creative` | 切换至创造服 |
| `/server main` | 切换至生存服 |
| `!s` | 切换旁观模式（英文感叹号） |
| `#消息内容` | 与 QQ 群互通聊天（每句话前加 `#`） |

**示例：**`#服务器现在有人能帮我备货吗`

---

## QQ 群绑定

首次进入服务器后，需在 QQ 群内发送以下消息完成绑定：

```
绑定 服务器验证码
```

> 验证码在首次进入服务器时会通过对话框提示，完成绑定后即可实现 QQ 群与服务器聊天互通。
