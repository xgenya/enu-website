# 文档简介

欢迎来到 **ENU Website** 的文档中心。

本文档提供了关于如何使用和扩展本平台的详细说明。

## 内容结构

- **快速入门** — 从零开始搭建环境并运行项目
- **配置** — 了解如何自定义站点配置

## 关于本项目

ENU Website 基于以下技术栈构建：

| 技术 | 版本 | 说明 |
|------|------|------|
| [Next.js](https://nextjs.org) | 16.x | React 应用框架 |
| [Nextra](https://nextra.site) | 4.x | 文档框架 |
| [React](https://react.dev) | 19.x | UI 库 |
| [TypeScript](https://www.typescriptlang.org) | 5.x | 类型安全 |
